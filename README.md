# 🎁 열쇠 모으기 게임

10×10 판(1~100번 칸) 안에 숨겨진 열쇠와 함정을 찾아 점수를 모으는 4팀 대전 보드게임

---

## 🗂️ 게임 모드

| 모드 | 설명 |
|------|------|
| **A 일반 모드** | 보물 위치 고정 (👻 투명=1·100번, 💣 폭탄=44·77번) |
| **B 스페셜 모드** | 아이템 개수를 직접 설정하고 랜덤 맵 생성 |

---

## 🔑 아이템 구성 (기본값)

| 아이템 | 개수 | 점수 | 설명 |
|--------|------|------|------|
| 🔑 일반 열쇠 | 25개 | +1점 | 기본 보물 |
| 🗝️ 특수 열쇠 | 5개 | +3점 | 희귀 보물 |
| ✨ 황금 열쇠 | 2개 | +5점 | 고가 보물 |
| 👻 투명 열쇠 | 2개 | +10점 | 게임 종료 후 공개, 역전 가능 |
| 🙈 함정 | 3개 | 없음 | 즉시 공개, 다음 턴 패스 |
| 💣 폭탄 | 2개 | -5점 | 발견자 -5점 + 주변 8칸 열쇠 파괴 |

> B 스페셜 모드에서는 각 아이템 개수를 자유롭게 설정할 수 있습니다.

---

## 🎲 진행 방법

1. 매 라운드 시작 전 미니게임(퀴즈, 인물 맞추기 등)을 진행
2. 미니게임 결과에 따라 1~4등 순서를 입력 (또는 🎲 랜덤 순서)
3. 순서대로 1~100번 중 원하는 칸을 선택
4. 15라운드 × 4팀 = 총 60번 선택 후 게임 종료

> **B 스페셜 모드** 전용 ⚡ **랜덤 선택** 버튼: 클릭 1회로 한 라운드 전체(4팀 랜덤 선택) 자동 진행

---

## 📌 칸 선택 결과

- **아이템이 있는 칸**: 해당 아이템 효과 즉시 적용
- **빈 칸**: 지뢰찾기처럼 주변 8칸 아이템 개수 표시
  - 숫자 진한 빨간색 = 주변에 특수 아이템(특수/황금/투명/폭탄) 있음!

---

## 💣 폭탄 규칙

- 발견한 팀: **-5점**
- 폭탄 주변 8칸에 이미 발견된 열쇠가 있으면 **모두 파괴**
  - 파괴된 열쇠 소유 팀은 해당 열쇠 점수만큼 추가 감점
  - 일반(-1) / 특수(-3) / 황금(-5) / 투명(-10)

---

## 👻 투명 열쇠 규칙

- 획득해도 게임 중에는 공개되지 않음
- 누가 획득했는지는 게임 종료 후 결과 공개 단계에서 확인
- 개당 +10점으로 역전 가능

---

## 🏁 결과 공개 순서

게임 종료 후 **다음 ▶** 버튼을 누를 때마다 단계별 공개:

1. 📦 보물 개수 공개 (일반 열쇠)
2. 🗝️ 특수 열쇠 공개
3. ✨ 황금 열쇠 공개
4. 👻 투명 열쇠 공개 (룰렛 연출)
5. 💣 폭탄 공개 (룰렛 + 주변 💨 연기 연출)
6. 🏁 최종 결과 확인

---

## ⚙️ 기타

- 같은 칸 중복 선택 불가
- 🙈 함정을 밟은 팀은 다음 라운드 선택 패스
- 🏠 처음으로 버튼으로 언제든지 첫 화면으로 복귀 가능
