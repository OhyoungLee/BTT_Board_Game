🎮 열쇠 모으기 게임 설명
10×10 판(1~100번 칸) 안에 숨겨진 열쇠를 찾아 점수를 모으는 게임

🔑 열쇠 구성
- 일반 열쇠 35개 : +1점
- 특수 열쇠 5개 : +3점
- 황금 열쇠 2개 : +5점
- 투명 열쇠 2개 : +10점

총 44개의 열쇠가 숨겨져 있음

🎲 진행 방법
1. 매 턴 퀴즈, 인물 맞추기 등 미니게임을 진행
2. 미니게임 결과에 따라 1~4등 순서가 정해짐
3. 순서대로 1~100번 중 원하는 칸을 선택

📌 칸 선택 결과
- 열쇠가 있으면 해당 열쇠를 획득하고 점수를 얻는다
- 열쇠가 없으면 지뢰찾기처럼 주변 8칸에 있는 열쇠 개수가 공개
예시) "주변 열쇠 4개"

👻 투명 열쇠
- 획득해도 공개되지 않는다
- 누가 획득했는지는 게임 종료 후 공개됩니다.
- 개당 +10점으로 역전 가능성

🏁 게임 종료
- 각 팀 15턴씩 진행 (총 60턴)
- 종료 후 획득한 열쇠 점수를 합산
- 최종 점수가 높은 순서대로 1~4등을 결정

※ 같은 칸은 중복 선택 불가
※ 투명 열쇠를 제외한 획득 열쇠 및 점수는 공개
